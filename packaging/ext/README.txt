setuptools 2.2
jsonpointer 1.1
six 1.7.2
ply 3.4
decorator 3.4.0
jsonpatch 1.3
jsonpath-rw 1.3.0
recordtype 1.1
urlparse2 1.1.1
pyreadline 2.1
validictory 1.0.1
python-ilorest-library 1.9.0

Then place it in this folder. This is primarily tested with Python 2.7.11.
